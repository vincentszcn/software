Using Tokenmon
--------------

Start Tokenmon.exe from its home directory. Complete usage 
instructions are available in the on-line help file.

Building Tokenmon
-----------------

Regmon consists of a device driver and a GUI. The driver was built with
the Windows NT 4 DDK Build environment and the GUI was compiled with 
Microsoft Visual C++ 6.0.  

The help was entered with Microsoft Word 97 and Help Workshop 
for Word 97.

To install, copy Tokenmon.exe and Tokenm.sys to the same 
directory.
